# Style-Bazaar
StyleBazaar is a modern fashion e-commerce web app developed using HTML, CSS, and JavaScript. It features product filtering, search, cart system, and a complete checkout flow.
# Features
• Responsive fashion e-commerce website UI
• Product listing with category and type filters
• Search functionality for products
• Sort products by price and customer rating
• Shopping cart with add/remove item functionality
• Dynamic cart total, discount, and shipping calculation
• Modal-based cart and checkout interface
• Multiple payment options (Card, UPI, Net Banking, Cash on Delivery)
• Checkout form with user input validation
• Order summary and successful order confirmation
• Clean and modern user interface design
# Technologies Used
• HTML5 – Structure and layout of the website
• CSS3 – Styling, responsive design, and animations
• JavaScript (ES6) – Dynamic content, cart logic, filtering, and form handling
• DOM Manipulation – Real-time UI updates
• Responsive Web Design – Mobile-friendly layout
